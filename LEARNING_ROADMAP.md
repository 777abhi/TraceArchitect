# Agentic AI & Generative AI Test Orchestration — Learning Roadmap

This roadmap guides a learner through a focused, 30-minute daily path to master **Agentic AI & Generative AI Test Orchestration**. Over four weeks, you will move from setting up a local environment and understanding LLM foundations for QA, to building self-healing tests, evaluating non-deterministic AI systems, and finally orchestrating autonomous QA agents in CI/CD with production telemetry.

## Rules for Engagement

- **Daily lessons are strictly 30 minutes**, split as:
  - 10 minutes — theory
  - 15 minutes — practical hands-on task
  - 5 minutes — reflection
- **Day 1 always starts with a step-by-step local environment setup** before any advanced concepts are introduced.
- **Wait for the learner's confirmation after each day's task** before moving on to the next day.

## Roadmap Structure

### Week 1: Environment Setup & LLM Foundations for QA

- **Day 1 — Local Setup:** Python 3.11+, Playwright, VS Code, and API key setup (or a local Ollama install).
- **Day 2 — Structured Outputs & JSON Mode** for test case generation.
- **Day 3 — Function Calling & Tool Use** in test automation.
- **Day 4 — Parsing DOM Trees & Accessibility Trees** for LLM context.
- **Day 5 — Week 1 Synthesis:** build a script that generates Playwright tests from user stories.

### Week 2: Agentic Architecture & Self-Healing Tests

- **Day 6 — The Agentic Loop:** Observe → Orient → Decide → Act in Quality Engineering.
- **Day 7 — Capturing Execution Failures & DOM Snapshots** on error.
- **Day 8 — Self-Healing Selectors:** prompting an LLM to repair broken locators.
- **Day 9 — Auto-retry and validation loops** with dynamic healing.
- **Day 10 — Week 2 Project:** build a self-healing Playwright test runner.

### Week 3: Testing AI Systems / Evaluating Non-Deterministic Models

- **Day 11 — Intro to AI Evaluation Frameworks:** DeepEval / Ragas.
- **Day 12 — Measuring Hallucinations, Faithfulness, and Answer Relevance.**
- **Day 13 — Red Teaming & Prompt Injection Testing** for QA.
- **Day 14 — Automated Evaluation Pipelines.**
- **Day 15 — Week 3 Project:** write an automated evaluation suite for a RAG application.

### Week 4: CI/CD Orchestration & Production Telemetry

- **Day 16 — Integrating AI Agents into GitHub Actions / GitLab CI.**
- **Day 17 — Shift-Right Testing:** feeding production logs back to the agent.
- **Day 18 — Dynamic Risk-Based Regression Suite Selection.**
- **Day 19 — Cost & Latency Optimisation** for test agents.
- **Day 20 — Final Capstone:** deploy an autonomous QA Agent pipeline.

## How to use this roadmap

Work through one day at a time, keeping each session to the 30-minute structure above. Complete the hands-on task, reflect on what you learned, and confirm completion before advancing. Use the checklist below to track your progress.
